<h1 align="center">Sistema SGC com DDD</h1>
<h3 align="center">Sistema de gerenciamento de clientes desenvolvida em C# usando a metodologia DDD.</h3>

<p align="left"> <img src="https://komarev.com/ghpvc/?username=thbarnabe&label=Profile%20views&color=0e75b6&style=flat" alt="thbarnabe" /> </p>
